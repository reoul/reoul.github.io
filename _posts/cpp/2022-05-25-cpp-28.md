# 반복자
- 어떤 컨테이너(자료구조)에 접근하든 동일한 방법으로 접근하기 위해 제공되는 객체
- 포인터라고 생각해도 된다

  ## 모든 요소 출력하기
  ```cpp
  for(std::vector<int>::iterator iter = scores.begin(); iter != scores.end(); ++iter)
  {
      std::cout << *iter << " ";
  }
  ```
