이 글은 C++ 참조(Reference)를 공부하고 정리한 글입니다
{: .notice--warning}

```cpp
int num = 47;
int& reference = num;

int& reference = NULL;    // error
int& reference;           // error
```
- `NULL`이 될수 없음
- 초기화중에 반드시 선언되어야 함
- 참조하는 대상을 바꿀 수 없음
- 우리가 소유하지 않은 메모리 장소를 가리킬 수 없음
- 함수의 매개변수가 Reference일 경우 유효한 값만 들어온다는 가정하에 로직을 짤 수 있음(안전한 프로그래밍이 가능해짐)
