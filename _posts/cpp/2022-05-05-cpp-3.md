이 글은 C++ 네임스페이스(namespace), using을 공부하고 정리한 글입니다
{: .notice--warning}

## namespace

```cpp
namespace blog
{
    void ShowName();
}

int main()
{
    blog::ShowName();    // 이런식으로 호출한다
}
```

- `네임스페이스` : 함수, 클래스 같은 것들과의 **이름 충돌을 피하기 위해**

## using

```cpp
using namespace std;

int main()
{
    cout << "hello world" << endl;
}
```

- `using` : 사용하면 `std::` 와 같이 함수 앞에 네임스페이스를 안 적어도 된다
